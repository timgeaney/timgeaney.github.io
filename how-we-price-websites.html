Pricing websites is tricky business but here's how we do it.

Most websites projects are made up of the following phases.

  1. 1. Design phase.
  2. 2. Convert designs to html, css and javascript template pages.
  3. 3. Make pages Mobile responsive.
  4. 4. Build out functionality.
  5. 5. Content Management System conversion.
  6. 6. SEO
  7. 7. Testing.

So we start with a site map. Whats a site map?

Thats a site map. It is basically a higher level diagram of what information your website will contain and how it will be orgainized. After completing your site map we will know how many web pages it will have, what are thoses pages functionality and what are their relationship to each other.

From the above site map we can see that there are 7 pages. But to complete this site we may only need to build out 6 templates. Because the Product 1 and Product 2 web pages are the same web template.

Whats a web template?

Web Templates are basically web pages that follow a distinct layout and design. The three images above are of three seperate web templates all following different layouts. The images below are of the same web template with a common header, sidebar and main content area.

How Come a 50 page website can cost the same as a 5 page one?

Ok so if the 5 page website uses 5 seperate web template designs @ €400.00 each that equals €2,000.00. If the 50 page website uses 3 template deisgns for all its 50 pages thats 3 x €400.00 which equals €1,200.00 plus €800.00 to set up the other 47 pages navigation and pages in wordpress. (note these figures are just approx. and are for illustration purposes only. They do not include entering content etc.) So thats how a 5 page website can cost the same as a 50 page one. Clear as mud! If you look at thesolar adtek website you can see that this website has 8 distinct templates.
