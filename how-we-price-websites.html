Pricing websites is a tricky business but here's how we do it.

First we listen to what your project goals and requirements are then we breakdown the project in logical chunks. Most websites projects are made up of the following phases.

  1. 1. Design.
  2. 2. Convert designs to html and css template pages.
  3. 3. Make pages Mobile responsive.
  4. 4. Build out functionality.
  5. 5. CMS conversion.
  6. 6. SEO
  7. 7. Testing.

After we determine the project phase we then come up with a site map. What’s a site map?

That’s a site map. It is basically a higher-level diagram of what information your website will contain and how it will be organized. After completing your site map we will know how many web pages it will have, what are those pages functionality and what are their relationships to each other.

From the above site map we can see that there are 7 pages. But to complete this site we may only need to build out 6 templates. Because the Product 1 and Product 2 web pages are the same web templates.

Whats a web template?

Web Templates are basically web pages that follow a distinct layout and design. The three images above are of three separate web templates all following different layouts. The images below are of the same web template with a common header, sidebar and main content area.

When pricing your project we assign a web template to each site map page some pages will share the same template therefore reducing the price.

Design

Once we have a clear picture of your requirements we build out a one page design this gives us an accurate depiction of what the finally website will look like and allows the client to come on board and give feedback that we can then alter the design to reflect.

Usually designs evolve over the course of the project as new requirements or content is added or omitted.

Convert to html & css

We have our site map and we know how many templates are required so now all we need is your content. Once you give us this we start building the site out in html5/css & sass. At this stage of the process the client has input into the design and any changes they want made are made here.

Functionality

Now Frontend Functionality gets built in. Frontend Functionality are all the JavaScript elements of the project that need to be build. For example JavaScript carousels/ image gallery, sliders etc.

CMS

CMS and server side functionality. We use Wordpress as our CMS of choice it powers approx. 25% of the web and for good reason. Huge sites like TechCrunch, BBC America, MTV News, Sony Music all use Wordpress. WordPress is an awesome content management for websites of all shapes and sizes.

SEO

We perform a full search engine optomization for every page. For a complete list of seo services.

Testing

Then, a very important stage, everything gets tested to make sure its running smoothly and security plugins are set up and analytics configured.

Summary

We want our pricing to be transparent and easy for you to understand so every project we quote for will give you a detailed breakdown that includes costs for the above steps.
